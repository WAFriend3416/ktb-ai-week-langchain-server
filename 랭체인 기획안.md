@기획서.md 
    에서 언급된 AI 분석을 진행하기 위해 프롬프팅을 단계적으로 실행할 수 있는 랭체인 시스템이 필요함.

해당 랭체인 시스템은 작성된 프롬프트를 실행시키고 그 결과값을 저장하는 과정을 맡아야함.

회사쪽 
1. 회사의 컬처핏을 분석하기 위해 데이터를 수집하는 프롬프트를 작동 시킨다.
    - 입력 값 : (최소기준) 채용공고, 회사 공식 페이지(인재상 같은걸 얻어내기 위해)
        - 추후 버전이 업그레이드 되면서 더 많은 정보를 가져올 수 있도록 프롬프트를 개선해나갈 예정.
        - 현재는 데이터 수집 프롬프트를 작동 시킨다 정도만 진행되면 됨.
2. 1번의 프롬프트의 결과를 회사의 컬처핏을 분석하는 프롬프트에 넣고 작동 시킨다.
    - 입력 값 : 1번 프롬프트의 결과(텍스트)
3. 2번 프롬프트의 결과를 저장한다.
    - 추후 동일 또는 유사한 회사의 컬처핏 분석이 요구됐을때 해당 자료 재사용이 목적

구직자쪽
1. 구직자의 이력서/포트폴리오를 분석하는 프롬프트를 작동 시킨다.
    - 입력 값 : 구직자의 이력서/포트폴리오

컬처핏 비교
1. 위 프롬프트들로 생성된 회사와 구직자의 컬처핏을 비교하는 프롬프트를 작동 시킨다.
    - 입력 값 : 회사 컬처핏 분석 결과, 구직자 이력서/포트폴리오 분석 결과
