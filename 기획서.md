## **1. 문제 정의**

1. 해결하려는 문제: 

- 회사 인재상을 미리 파악하면 자소서/면접에서 유리한 고지 선점 가능
- 하지만 현실에서는 취준생이 회사의 인재상과 컬쳐핏을 파악하기 어려움
- 회사 정보가 여러 플랫폼에 분산되어 있어 수집이 번거로움
- 데이터 형식이 제각각이라 수집 + 정제 과정이 필요
- 수치화되지 않은 정보로 객관적 판단이 어려움
1. 타깃 사용자: 
- 개발자 취준생 (나랑 맞는 회사를 찾고 싶은 사람들)
1. 문제의 중요성: 
- 기업 84.7%가 1년 이내 조기퇴사자 경험, 퇴사 사유 3위가 '조직문화 불만족'(31.5%) - 사람인 조사
- 독성 조직문화는 연봉보다 이직에 10.4배 더 큰 영향 → 컬쳐핏이 연봉보다 중요 - SHRM 2024
- 직장인 66%가 입사 후 컬쳐핏 불일치 경험, 그 중 절반이 6개월 내 퇴사 → 사전 파악이 핵심
- 채용 담당자 84%가 컬쳐핏을 핵심 채용 기준으로 선정 → 구직자도 이를 알아야 유리

**유사 서비스 / 경쟁 서비스 분석(1~2개)**

- 잡플래닛 / 장점: 회사 리뷰 정보 풍부 / 한계: 커리어 매칭 기능 부족
- 캐치 / 장점: 채용 정보 통합 / 한계: 커리어 분석 기능 제한적
- 링크드인 / 장점: 회사 활동 정보 / 한계: 개인화된 매칭 부족
- [Jobcompass.ai](http://Jobcompass.ai) / 장점 :  이력서 분석 및 피드백, HR 파트 컨택 기능/ 한계 : 서비스 초기, Linked In에 의존적

**기술적 차별성(남들과 다른 기술적 강점 또는 시도)**

- 경쟁 서비스 방식 대비 새로운 점:
- AI가 회사 정보(공고, 기업 사이트, 연혁, 인재상)를 **자동 수집/분석**
- LLM이 공식을 만들어 컬쳐핏 점수를 단계로 수치화
- 구직자 이력서/포트폴리오 분석을 통한 양방향 매칭
- 한 번에 여러 소스(기술 블로그, 유튜브, 리뷰 사이트) 통합 분석

## **3. 핵심 기능 정의**

**서비스의 핵심 컨셉 한 줄 요약**

- "나랑 맞는 회사를 찾고 싶은 개발자 취준생을 위해 AI 기반 컬쳐핏 분석을 통해 회사-구직자 매칭 문제를 해결하는 서비스"

• **“개발자 취준생이 나와 맞는 회사를 찾을 때 가장 막막한 ‘컬쳐핏 파악’ 문제를 AI로 쉽게 풀어주는 서비스”**
• **“취업 준비 과정에서 시간 많이 들고 기준도 모호한 ‘컬쳐핏 탐색/검증’ 과정을 AI로 간단하게 만들어주는 서비스”**

**이번 해커톤에서 구현할 핵심 기능**

- 핵심 기능 1
    
    ### 기능명
    
    - 회사 컬쳐핏 분석 시스템
    
    ### 기능 설명
    
    - 회사 정보(공고, 기업 사이트, 연혁, 인재상)를 AI가 자동 수집하여 기업 컬쳐핏 요약 생성
    
    ### 사용자 가치
    
    - 여러 사이트를 돌아다닐 필요 없이 한 번에 회사 컬쳐핏 정보 확인 가능
    
- 핵심 기능 2
    
    ### 기능명
    
    - 구직자 컬쳐핏 테스트
    
    ### 기능 설명
    
    - 구직자 이력서/포트폴리오를 AI가 분석하여 회사-구직자 컬쳐핏의 테스트 결과 제공
    
    ### 사용자 가치
    
    - 나와 적합한 회사인지 단계로 객관적 판단 가능
    - 정성적인 컬쳐핏을 정량적인 단계로 확인 가능
    

**예상되는 기대효과**

- 서비스 사용 전후 변화
- 전: 여러 사이트를 돌아다니며 수동으로 회사 정보 수집 → 후: AI가 자동으로 통합 분석
- 직접적 이득: 회사 커쳐핏 정보를 한 번에 확인, 나와 적합한지 객관적 지표로 판단 가능
- 시너지: 정보 수집 시간 절약, 더 많은 회사 비교 가능, 취업 준비 효율성 향상
